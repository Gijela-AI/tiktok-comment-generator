### How to Comment on TikTok Pictures

Commenting on TikTok pictures is simple:
1. Open the TikTok app and find the picture post you want to comment on
2. Tap the comment icon on the right side of the screen
3. Type your comment in the input box at the bottom
4. Tap the send button

### How to Pin TikTok Comments

If you're a video creator, you can pin important comments:
1. Find the comment you want to pin in your video's comment section
2. Long press the comment
3. Select the "Pin Comment" option
4. The comment will appear at the top of the comment section
